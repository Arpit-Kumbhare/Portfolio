# Arpit Kumbhare Portfolio

Modern, minimal React (Vite) portfolio featuring a hero intro card, sticky top navbar with a simple greeting, and a centered floating dock for rapid section navigation.

## Features
- Hero card introducing name, tagline, summary
- Bottom-centered blurred dock with icons linking to About, Projects, Experience, Contact
- Responsive grid for projects & timeline layout for experience
- Simple theming using CSS variables

## Getting Started
```bash
npm install
npm run dev
```
Visit: http://localhost:5173

## Updating Content
Edit `src/data/content.js` to replace placeholder bio, projects, experience and contact info with real resume details.

## Build
```bash
npm run build
npm run preview
```

## Folder Structure
```
src/
  components/  React UI components
  data/        Centralized content
  styles.css   Global styles
```
